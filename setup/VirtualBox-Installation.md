# VirtualBox Installation

## Objective

Install Oracle VirtualBox to host the Windows 10 and Kali Linux virtual machines used in the SOC Detection Lab.

---

## Prerequisites

- Windows 11 Host
- Administrator privileges
- Internet connection

---

## Steps

1. Download Oracle VirtualBox.
2. Install VirtualBox using the default options.
3. Install the VirtualBox Extension Pack.
4. Restart the host machine if required.
5. Launch VirtualBox.

---

## Verification

- VirtualBox opens successfully.
- The VirtualBox Manager dashboard is displayed.
- No installation errors are shown.

---

## Troubleshooting

| Issue | Solution |
|--------|----------|
| VirtualBox won't start | Enable virtualization (VT-x/AMD-V) in BIOS/UEFI. |
| Installation blocked | Run the installer as Administrator. |
| Virtual machines fail to start | Check virtualization settings and Windows Hyper-V configuration. |
